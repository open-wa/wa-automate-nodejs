# Type Alias: ChatId

> **ChatId**: [`ContactId`](/docs/reference/api/model/aliases/type-aliases/ContactId) \| [`GroupChatId`](/docs/reference/api/model/aliases/type-aliases/GroupChatId)

A chat id ends with `@c.us` or `@g.us` for group chats.

Example:

A group chat: `"447123456789-1445627445@g.us"`
A group chat: `"447123456789@g.us"`
