# @open-wa/cli-app

WhatsApp CLI - Easy API mode for @open-wa/wa-automate

Part of the [@open-wa v5 monorepo](https://github.com/open-wa/wa-automate-nodejs).

## Development

- `pnpm --filter @open-wa/cli-app build`
- `pnpm --filter @open-wa/cli-app dev`
- `pnpm --filter @open-wa/cli-app lint`
- `pnpm --filter @open-wa/cli-app start`
