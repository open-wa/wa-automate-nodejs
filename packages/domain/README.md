# @open-wa/domain

Domain data structures for open-wa WhatsApp automation (Collectors, etc.)

Part of the [@open-wa v5 monorepo](https://github.com/open-wa/wa-automate-nodejs).

## Install

```bash
pnpm add @open-wa/domain
```
