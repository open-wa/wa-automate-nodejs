# @open-wa/screencaster

Headless browser screencasting for open-wa — treeshakable server + client

Part of the [@open-wa v5 monorepo](https://github.com/open-wa/wa-automate-nodejs).

## Install

```bash
pnpm add @open-wa/screencaster
```
