# Switchboard

This folder contains workflow artifacts — review outputs, handoff logs, and audit reports.

See `WORKFLOW_REFERENCE.md` for full workflow documentation.

### Quick Start
- Terminal and messaging setup is handled automatically on extension activation.
- Use `/handoff` to delegate tasks to other agents.
- Use `/improve-plan` for plan hardening plus adversarial review.
- Use `/challenge` for internal adversarial review without delegation.
