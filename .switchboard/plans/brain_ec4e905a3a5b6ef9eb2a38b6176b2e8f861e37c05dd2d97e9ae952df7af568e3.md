# Implementation Plan - v5 Post-QR Readiness Recovery

This plan implements the fixes for the v5 session persistence regression where the client fails with `store_missing` after a QR scan.

## User Review Required

> [!IMPORTANT]
> This plan follows the "Option C" mechanism approved in the issue notes: adding an inline polling gate in `createClient.ts` to ensure `Store.Msg` is present before patching.

## Proposed Changes

### Core Bootstrap Logic

#### [MODIFY] [packages/core/src/createClient.ts](file:///Users/Mohammed/projects/tools/wa/packages/core/src/createClient.ts)
- Insert a 15-second `Store.Msg` polling gate after post-auth validation but before marking readiness as `satisfied`.
- Handle timeout by logging diagnostic `Store` keys and emitting a `bootstrap.store_settle` fatal error.
- Ensure `runtime_activation_ready` is only emitted after the store is confirmed.

#### [MODIFY] [packages/core/src/transport/Transport.ts](file:///Users/Mohammed/projects/tools/wa/packages/core/src/transport/Transport.ts)
- Update `recoverRuntimeForCurrentDocument()` to account for store integrity in recovery decisions, specifically for the second reconciliation call.
- Re-evaluate post-auth stabilization branching to ensure fresh-auth flows are correctly handled.

### Testing

#### [NEW] [packages/core/test/unit/bootstrapContract.test.ts](file:///Users/Mohammed/projects/tools/wa/packages/core/test/unit/bootstrapContract.test.ts)
- Add regression tests for:
    - Post-auth runtime with missing store (must not be ready).
    - Store-settle gate timeout (must fail with diagnostics).
    - Store-settle gate delayed success (must proceed).
    - Post-auth recovery path when store is missing.

### Telemetry

#### [MODIFY] [packages/wa-automate/src/cli-runtime.ts](file:///Users/Mohammed/projects/tools/wa/packages/wa-automate/src/cli-runtime.ts)
- Add human-readable narration for post-auth reconciliation and store-settle gate progress.
- Improve patch application logging (source, tag, timing).
- Add clear diagnostic summaries on failure.
